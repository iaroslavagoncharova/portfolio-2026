## Portfolio website

## Overview

A modern, responsive portfolio website built with **Next.js** and **React** to display projects, work experience, and technical skills.

---

## Features

* Single-page layout with smooth section navigation
* Projects showcase with descriptions, tech stack, and external links
* Experience timeline grouped by year and role
* Contact form with animated feedback
* Animations with Framer Motion for a modern user experience
* Fully responsive design
* The website is deployed with Vercel

## Getting Started

### Local Development

```bash
git clone https://github.com/iaroslavagoncharova/portfolio-2026.git
npm install
npm run dev
```

Open `http://localhost:3000` in the browser.
